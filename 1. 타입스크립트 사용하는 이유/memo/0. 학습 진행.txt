타입스크립트 학습 진행
- 본 강좌와 함께 캡틴판교님 블러그 "타입스크립트 핸드북"과
  함께 병행하기 

- 강의에서는 타입스크립트를 따로 설정하지 않고 
  TypeScriptlang 라는 사이트에서 진행하기 때문에 
  여기에 타입스크립트 코드를 작성하는 것이 어렵다.

- 따라서 별도로 vscode에 타입스크립트를 사용할 수
  있게 설정한다. 

- 각 code 폴더에는 각 챕터에 해당하는 .ts/.js(선택)을 
  담아서 관리
  → 각 ts 파일에 함수를 구현하면 중복된 함수라고 뜨는데 
    이건 tsconfig 파일이 없기 때문이다. 
  → 또는 한 폴더에 동일한 함수를 사용할 때에도 발생함

- 컴파일 하기 
 ● 해당 프로젝트 폴더에서 (여기서는 ts-example) 아래와 같이 입력 
  ◎ tsc
   
 ● 그러면 dist폴더에 자바스크립트로 변환된 파일이 담긴다. 
   이 자바스크립트 파일을 실행하면된다. 

- ts-node 사용하기 
 ts-node는 타입스크립트로 작성된 (아직 컴파일되지 않는) 파일을 
 콘솔 환경에서 실행하는데 도와준다. 
 
 설치 
 yarn add -g ts-node
 → 명령어를 사용하기 위해 글러벌로 설치

 실행하기
 → ts-node 실행할파일명.ts(확장자는 생략 가능)
